Quick Implementation

Integrate the NewsScope API into your application using Axios. Ensure you include your secret key in the headers.

const axios = require('axios');

await axios.get('https://api.newsscope.com/v1/news', {
  headers: {
    'X-API-KEY': 'YOUR_SECRET_KEY'
  }
});
